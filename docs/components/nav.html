Nav

Base navigation component for links, tabs, and pills. Tab switching is wired by vitrus.bundle.min.js via data-vitrus-toggle="tab".

Tabs

Use data-vitrus-toggle="tab" and data-vitrus-target on each trigger. Arrow keys move focus between tabs when the list has role="tablist".
