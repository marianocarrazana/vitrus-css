Theming

Themes & dark mode

VitrusCSS exposes every design token as a CSS custom property. Switch presets at runtime with data-vitrus-theme on <html> — no rebuild required.

Activation

Vitrus uses data-vitrus-theme instead of Bootstrap's data-bs-theme. Set the attribute before paint to avoid a flash, or use the sidebar picker on these docs. 

<html data-vitrus-theme="midnight">
  ...
</html>

On first visit, when no theme is stored, the docs honor prefers-color-scheme: dark and apply the midnight preset automatically.

Built-in presets

Theme Base Blur Character
default Light glass 16px Baseline Vitrus look
frost Light glass 28px Heavy frost, high transparency
crystal Light glass 8px Crisp panels, minimal blur
midnight Dark glass 20px Dark-base glassmorphism

Custom overrides

Override any token in your own stylesheet after importing VitrusCSS.

:root {
  --vitrus-primary: #ec4899;
  --vitrus-glass-blur: 24px;
}

[data-vitrus-theme="midnight"] {
  --vitrus-glass-bg: rgba(2, 6, 23, 0.6);
}
