Components

Buttons

Use the .btn base class with a variant modifier to render a glass-tinted button. Works on <button>, <a>, and <input> elements.

Variants

Eight semantic variants. Each tints a translucent surface with the variant color. 

<button type="button" class="btn btn-primary">Primary</button>
<button type="button" class="btn btn-secondary">Secondary</button>
<button type="button" class="btn btn-success">Success</button>
<button type="button" class="btn btn-danger">Danger</button>
<button type="button" class="btn btn-warning">Warning</button>
<button type="button" class="btn btn-info">Info</button>
<button type="button" class="btn btn-light">Light</button>
<button type="button" class="btn btn-dark">Dark</button>
<button type="button" class="btn btn-link">Link</button>

Outline

Transparent fill, colored border, and a colored hover state.

<button type="button" class="btn btn-outline-primary">Primary</button>
<button type="button" class="btn btn-outline-success">Success</button>

Sizes

<button type="button" class="btn btn-primary btn-sm">Small</button>
<button type="button" class="btn btn-primary">Default</button>
<button type="button" class="btn btn-primary btn-lg">Large</button>

Block

<button type="button" class="btn btn-primary w-100">Block button</button>

Disabled

Add the disabled attribute on a real <button> or the .disabled class on <a>.

Disabled link 
<button type="button" class="btn btn-primary" disabled>Disabled</button>
<a href="#" class="btn btn-primary disabled" role="button" aria-disabled="true">Disabled link</a>

Close button

For dismissing alerts and modals. JS behavior ships in Phase 3; the markup is in place now.

← on dark backgrounds, use .btn-close-white 
<button type="button" class="btn-close" aria-label="Close"></button>
