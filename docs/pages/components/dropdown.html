Overlays

Dropdown

Toggleable glass menus positioned with Popper. Use data-vitrus-toggle="dropdown" instead of Bootstrap's data-bs-toggle.

Basic example

  • Action
  • Another action
  • Something else
<div class="dropdown">
  <button class="btn btn-secondary dropdown-toggle" type="button"
    data-vitrus-toggle="dropdown" aria-expanded="false">
    Dropdown button
  </button>
  <ul class="dropdown-menu">
    <li><a class="dropdown-item" href="#">Action</a></li>
  </ul>
</div>
