Introduction

Bootstrap 5 compatible.
Glassmorphism by default.

VitrusCSS is a Sass-authored component library that mirrors the Bootstrap 5 class API and ships translucent glassmorphism as the default look on every element. Drop in a stylesheet, write the markup you already know, and your UI gets frosted.

Light surface

Translucent white with a hairline border and soft shadow.

Tinted primary

The brand color tints a frosted layer. No solid fills here.

Success state

Contextual tinting for every semantic variant.

Warning

Warm tints stay readable; text contrast is preserved.

Quick start

Link the CSS bundle for styling. Add the JS bundle when you need interactive components (alerts, collapse, tabs, and Phase 3 overlays).

Via compiled CSS + JS

<link rel="stylesheet" href="dist/vitrus.min.css">
<script src="dist/vitrus.bundle.min.js" defer></script>

Vitrus uses data-vitrus-toggle, data-vitrus-target, and data-vitrus-dismiss instead of Bootstrap's data-bs-* attributes. Class names and HTML structure match Bootstrap 5.

Via Sass (recommended)

Override tokens before importing the entry. The library exposes every value on :root as a CSS custom property too, so you can theme at runtime. 

// Override Sass variables
$primary: #6366f1;
$glass-blur: 16px;

@import 'vitrus-css/css/vitrus';

Install

npm install vitrus-css

Themes

Four built-in presets ship with the library. Set data-vitrus-theme on <html> or use the header theme picker to switch live — glass blur, transparency, and base color update without a rebuild.

Theme Base Blur Character
default Light glass 16px Baseline Vitrus look
frost Light glass 28px Heavy frost, high transparency
crystal Light glass 8px Crisp panels, minimal blur
midnight Dark glass 20px Dark-base glassmorphism

Glass tokens

The values below are the heart of the Vitrus look. Every surface — buttons, cards, alerts, inputs — composes them into a frosted glass panel. They are exposed as CSS custom properties and overridden per theme.

Token Default Role
--vitrus-glass-bg rgba(255, 255, 255, 0.55) Translucent fill
--vitrus-glass-border rgba(255, 255, 255, 0.6) Hairline edge
--vitrus-glass-blur 16px Backdrop blur radius
--vitrus-glass-shadow 0 8px 32px rgba(15, 23, 42, 0.12) Soft, layered depth
